# Модуль транспортных моделей для APIv2

Содержит мультиплатформенные транспортные модели. Полностью изолирует все вопросы сериализации и десериализации от
потребителей. Помимо транспортных моделей, потребителю доступны только 4 метода:
1. `apiV2RequestSerialize` сериализует запрос объект в json-строку
2. `apiV2RequestDeserialize` десериализует запрос объект из json-строки
3. `apiV2ResponseSerialize` сериализует ответ объект в json-строку
4. `apiV2ResponseDeserialize` десериализует ответ объект из json-строки
